Parser

A parser converts a sequence of tokens into a tree data structure. This tree can then be used for analyzing the program.

Example

Here is the PL/1 syntax tree for the multiply program:

$ ./pl1_parser.py < examples/multiply.pl1 
PROGRAM
  BLOCK
    VARIABLES
      ('NAME', 'x')
      ('NAME', 'y')
      ('NAME', 'z')
    BEGIN
      SET
        ('NAME', 'x')
        EXPRESSION
          TERM
            ('NUMBER', 10)
      SET
        ('NAME', 'y')
        EXPRESSION
          TERM
            ('NUMBER', 20)
      SET
        ('NAME', 'z')
        EXPRESSION
          TERM
            ('NAME', 'x')
            TIMES
              ('NAME', 'y')
      PRINT
        EXPRESSION
          TERM
            ('NAME', 'z')

Source Code Visualisation

We can also visualise programs using graphviz:

-- You will need to install graphviz in order to generate graphs from source code files.
$ ./pl1_graphviz.py < examples/multiply.pl1
